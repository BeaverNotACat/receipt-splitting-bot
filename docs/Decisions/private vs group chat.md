## В личных сообщениях, через гиперлинк
Кредитор отправляет ссылку на бота друзьям. При переходе по ссылке у каждого открывается чат с ботом, где пользователи гут параллельно работать над чеком

**Плюсы:**
- Есть доступ с BotKeyboard, и индивидумальным состояниям для каждого пользователя. Это даёт вести за ручку при регистрации, добавлении вирутального должника, переключении между чеками.
- Пользователи могут действовать независимо. Кредитор может через лс открыть какой-то старый чек, чтобы скопировать оттуда инфу, сколько ему должен человек без надобности писать команды в группу.
- Контекст ллм не захламляется попутными обсуждениями, в лс боту будут излагатсья только сухие факты кто что "ел"
**Минусы:**
- Не можем читать что другие пользователи натыкали в боте
- Если сидим не лично, нужно будет переключаться между чатами, чтобы обсуждать.

## В группе, через добавление туда бота
Кредитор может пригласить бота в заранее созданный вами груповой чат, где опользователи будут совместно работать над чеком.

**Плюсы:**
- Мы можем читать общения всех в боте
- Всё в одном месте
**Минусы:**
- Нет клавиатуры и индивидуальных состояний. Пользователи могут пользоваться только атомарными операциями через команды. Они заранее должны знать что за чем идёт: Сначала команда чтобы ввести имя и тд. Иначе будет происходить долбание лбом о стену, где человек вводит в команду, Бот выдаёт сообщение об ошибке, что  сначала нужно было ввести другую
- В контекст к ллм будут капать конфузящие выяснения кто всё таки взял сырную тарелку.
- Когда нам останавливаться, чтобы не всирать токены?

Реализация:
1. *Пригласили*
2. /register
3. /create_receipt
4. /join
5. *диалог*
6. /show_bills
7. /end
